# Medicare_Optimization_Project
This project analyzed 2M+ CMS service records (2021–2023) to identify drivers of reimbursement efficiency for Medicare Providers Texas, USA.
# Data Source
The Medicare data files (2021-2023) are available for download: Medicare Dataset - Google Drive
Place Medicare_Data_2021.csv, Medicare_Data_2022.csv, and Medicare_Data_2023.csv in the project folder before running the analysis.
# Place the following files in the project folder:
•	Medicare_Data_2021.csv
•	Medicare_Data_2022.csv
•	Medicare_Data_2023.csv
•	analysis_data.rds (processed analysis data)
The .rds file contains cleaned/processed data used for the optimization analysis.
